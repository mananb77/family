# The Bhargava Family — Memorial Site

A family memorial site honoring the Bhargava family, built with love and remembrance.

## Live Site

[mananb77.github.io/family](https://mananb77.github.io/family/)

## About

This site serves as a central hub connecting the individual memorial pages for our grandparents — Nana, Nani, Dada, and Dadi — while also showcasing the extended family tree across both maternal and paternal sides.

### Features

- **In Memoriam** — Memorial cards for all four grandparents with links to their individual sites
- **Family Tree** — Core family tree showing grandparents, parents, and children
- **Extended Family** — Two-sided layout (Maternal | Paternal) with all siblings and their families
- **Bilingual** — English and Hindi throughout
- **Responsive** — Fully responsive across desktop, tablet, and mobile

## Tech Stack

- React 19 + Vite
- Deployed via GitHub Actions to GitHub Pages
